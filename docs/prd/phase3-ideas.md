# Linear Lite - Phase 3 核心轻量管理能力 (Project & Task Management) 规划方案

## 重新定位与目标调整

基于产品“轻量级项目/任务管理系统”的本质定位，我们修正了演进方向。对于轻量级工具而言，**沟通协作（如评论、动态流）并不是真正的刚需**，用户完全可以在现有的 IM 工具中完成讨论，而在本系统中专注于状态流转。

因此，Phase 3 的核心使命是**补齐真正意义上的“管理”能力**，解决当任务量和项目量上升时的效率瓶颈：

1. **项目管理能力**：打破只能查看预置项目的限制，实现项目的完整生命周期管理。
2. **任务视图升级**：打破单一 Kanban 视图在海量任务下的局限性。
3. **时间管理维度**：引入任务的时间线概念，让“轻量”不等于“信息残缺”。

---

## Phase 3 核心功能模块

### 1. 结构管理：项目的新建与维护 (Project Management)

- **痛点**：目前左侧边栏的 Project 只能切换不能管理，无法支持真实的团队日常结构变动。
- **功能点**：
  - **新建项目 (Create Project)**：允许用户创建新项目，并自动生成类似 `ENG`、`PROD` 这样的三字母标识符 (Identifier)。
  - **项目设置页 (Project Settings)**：一个轻量的弹窗或页面，允许修改项目名称、标识符等基本信息。
- **价值**：让系统具备自我生长的能力，闭环了“项目隔离”的逻辑。

### 2. 效率升级：高密度列表视图 (List View)

- **痛点**：看板视图 (Board) 适合直观拖拽和观察当前迭代，但当某一列（如 Backlog）堆积几十个任务时，浏览效率极低，垂直空间浪费严重。
- **功能点 (参考 Linear)**：
  - 在主界面的右上角增加一个视图切换器（Toggle: Board / List）。
  - **列表视图 (List View)**：以高密度表格形式展示任务（展示 ID, Title, Status, Priority, Assignee, Due Date），支持紧凑排列。
  - **按状态/优先级分组**：即使在列表模式下，也可以保持数据的分组折叠能力（类似 Linear 的 Accordion 交互）。
- **价值**：真正满足“管理者”和“核心执行者”快速扫视、批量认知海量任务的刚需。

### 3. 时间维度：任务模型扩充 (Task Timeline Fields)

- **痛点**：目前只能看到创建和更新时间，无法管理“什么时候该做完”和“实际上什么时候做完”，导致无法衡量进度偏离。
- **功能点**：
  - **`dueDate` (预计完成时间 / 截止日期)**：允许在创建或编辑任务时设定。列表视图中可直观展示，若超期可飘红警告。
  - **`completedAt` (实际完成时间)**：系统级自动字段。当任务状态流转到 `Done` / `Canceled` 这一类终态时，后端自动打上时间戳。
- **UI 呈现**：在 `TaskCard` 上精简展示超期红点或截止日；在 `TaskEditor` 中增加优雅的日期选择器 (Date Picker)。
- **价值**：补齐任务三要素（人、事、时）中的最后一环。

## 落地演进路线建议

基于前后端分离的全栈架构，Phase 3 大约分为以下实施步骤：

1. **[Backend] 领域模型扩展**：
   - 数据库 `tasks` 表增加 `due_date` 和 `completed_at` 字段。
   - 实现 `Project` 的 POST / PUT 接口支持新建。
2. **[Frontend] Store 与模型对齐**：
   - 更新前端的 `Task` domain 类型，对接新的时间字段。
   - `projectStore.ts` 对接项目新建 Action。
3. **[Frontend] List View 开发**：
   - 提取原有 `BoardView.vue` 中的逻辑进行细分。
   - 引入一个全新的 `TaskListView.vue` 组件，并允许用户一键切换/保留视图偏好偏好记忆 (`localStorage`)。
4. **[Frontend] UI 扩充**：
   - 新增 `CreateProjectModal` 组件。
   - 在 `TaskEditor` 中集成轻量级的类 Linear DatePicker。

如果你觉得上述功能点（项目新建、Linear-style 列表视图、预计与完成时间）精准命中了你对第三阶段的管理述求，我们就按这个确立目标！
